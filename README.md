# Super Trunfo — Insetos 🦋

Jogo Super Trunfo com tema Insetos desenvolvido em Java SE.

## Tecnologias
- Java SE 17
- Java Swing (Interface gráfica)
- Arquitetura MVC
- MySQL (Banco de dados)
- MySQL Connector/J 5.1.26

## Como executar
1. Instale o Java JDK 17+
2. Instale o XAMPP e inicie o MySQL
3. Importe o banco: `sql/super_trunfo_insetos.sql`
4. Configure `resources/config.properties`
5. Execute `main/Main.java` no Eclipse

## Estrutura do Projeto
- `model/` — Classes de dados
- `view/` — Interface Swing
- `controller/` — Lógica do jogo
- `dao/` — Acesso ao banco de dados
